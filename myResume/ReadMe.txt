Resume Viewer
    This repository contains the code for a simple web application that allows users to view their resumes in PDF format. The application is implemented using HTML, CSS, and JavaScript.

How to Run
    To run the resume viewer locally on your machine, follow these steps:

    Clone the Repository:
        git clone https://github.com/...../resume-viewer.git
    Navigate to the Repository Directory:
        cd resume-viewer
    Open the HTML File:
        Open the index.html file in your preferred web browser.

Features
    Resume Preview: Uploaded resumes are displayed on the page for preview.
    Responsive Design: The resume viewer is responsive and adjusts its layout based on the device screen size.

Usage
    index.html: HTML file containing the structure and content of the resume viewer.
    styles.css: CSS file containing the styles for the resume viewer.
    script.js: JavaScript file containing the functionality for previewing resumes.
